
# ShipToNigeria

Live Demo: [https://seismic-assessment.isaiahozadhe.tech/](https://seismic-assessment.isaiahozadhe.tech/)

Welcome to ShipToNigeria! This project is a web application designed to help users ship items from various stores to Nigeria with ease. The app is built using Next.js and Tailwind CSS.

## Features
- Browse popular stores
- Learn how the shipping process works
- View platform highlights and services
- Get answers to frequently asked questions

## Setup Instructions

### 1. Prerequisites
- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### 2. Install Dependencies
Open your terminal and run:

```
npm install
```

### 3. Start the Development Server
To run the app locally, use:

```
npm run dev
```

This will start the development server. Open your browser and go to [http://localhost:3000](http://localhost:3000) to view the app.

### 4. Build for Production
To create an optimized production build, run:

```
npm run build
```

To preview the production build locally:

```
npm run start
```

## Project Structure
- `app/` - Main application pages and layout
- `components/` - Reusable UI components
- `public/` - Static assets like images
- `styles/` - Global styles (if any)

## Key Assumptions
- The app is for demonstration and informational purposes only.
- No real payment or shipping is processed through this app.
- All store and shipping information is for sample use.

## Customization
You can update images and text in the `public/images/` and `components/` folders to fit your needs.
